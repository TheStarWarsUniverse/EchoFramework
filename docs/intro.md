# About

Echo framework is a lightweight framework developed by [RandomMutiny](https://www.roblox.com/users/507966726/profile) and heavily inspired on [Knit framework](https://sleitnick.github.io/Knit/) by [sleitnick](https://www.roblox.com/users/308165/profile).

See the [Getting Started](https://thestarwarsuniverse.github.io/EchoFramework/docs/GetStarted) guide to start using Echo.
